# Flight Reservation and Management System

This project implements a database-driven flight reservation system developed using SQL and Java.

## Features
- Flight scheduling
- Passenger management
- Reservation system
- Database queries for flight operations

## Technologies
- Java
- SQL
- Relational Database Design

## Project Structure

database/  
Contains the SQL schema and queries.

src/  
Executable Java application (.jar).

report/  
Full project documentation.

## How to Run

1. Import the SQL file into MySQL or PostgreSQL
2. Run the `.jar` file
